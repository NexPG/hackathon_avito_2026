# Hackathon Avito 2026 — Microcategory Detection

Проект для автоматического анализа описаний объявлений на Авито с целью выделения микрокатегорий и определения отдельных услуг внутри комплексных работ.

## Описание проекта

Этот проект разработан в рамках хакатона **IT Purple Hack 2026**.

Основные задачи:
-  Выделение микрокатегорий из описаний объявлений
-  Определение, какие услуги можно выполнить отдельно
-  Генерация текста для таких услуг

Проект работает с объявлениями из категории:
> **«Ремонт и отделка»**

Более подробное описание:
 `IT_Purple_Avito.md`

---

##  Используемая модель

В основе проекта используется модель:
- deepvk/USER2-base

Проект ориентирован на обработку русскоязычных текстов.

## Как пользоваться

### 1. Подготовка данных
Файл: `dataset_preparer.ipynb`

Позволяет:
- Разделить датасет на train / test
- Очистить текст от спецсимволов и эмодзи
- Создать столбец `cleaned_description`

---

### 2. Загрузка моделей
Файл: `models_downloader.ipynb`

- Скачивание обученных моделей (если они доступны)

---

### 3. Обучение моделей
Файл: `model_trainer.ipynb`

- Обучение моделей на:
  - исходном датасете
  - пользовательском датасете

---

### 4. Оценка моделей
Файл: `evaluation_models.ipynb`

- Проверка качества моделей
- Работа с исходным или пользовательским датасетом

---

### 5. Использование на реальных объявлениях
Файл: `for_web-page_version.ipynb`

Позволяет:
- Обработать объявление с Авито
- Получить:
  - микрокатегории
  - список услуг, которые можно выполнить отдельно
  - сгенерированные тексты для этих услуг

 *Требуется Google Gemini API Key*

---

### 6. Результат работы
Файл: `rnc_test.csv`

- Результаты на rnc_test
